## 實現工具

1.  Vue-cli
2.  Vuex
3.  Vue-router
4.  Axios
5.  Bootstrap4
6.  Amazon S3
7.  高雄資料開放平臺

## Quickstart

```bash
# 下載專案
git clone https://github.com/jaw09/vue-spots-filter.git

# 安裝套件
npm install

# 在本地段瀏覽專案
npm run dev

# 將專案打包輸出
npm run build
```
